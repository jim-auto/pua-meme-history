PUA界隈ヒストリー

PUA / ナンパ / マッチングアプリ界隈の文化アーカイブ

Years

界隈の歴史

語源と黎明期

明治
「軟派」の誕生

「軟派（なんぱ）」は元々政治用語で「意見が軟弱な党派」の意味（1892年初出）。その後「異性との交遊を好む青少年の一派」（1909年）に転用。「硬派」の対義語として定着。

1970s
「ガールハント」から「ナンパ」へ

銀座に代わり新宿がアングラ文化の中心に。この頃「ガールハント」から「ナンパ」と呼ばれるように。1980年頃からカタカナ表記の「ナンパする」が定着。

オフライン: ストリート / クラブ / カーナンパ

1968
日本のディスコ文化の始まり

赤坂にMUGENがオープン、1971年に同ビルにビブロスが誕生。アメリカのディスコブームが日本に上陸し、ディスコが男女の出会いの場として機能し始める。

1980-84
第1次ディスコブーム / 六本木スクエアビル

別名「サーファーディスコブーム」。六本木スクエアビルは地下2階〜地上10階のうち10フロアがディスコという一大牙城に。原宿ではホコ天・竹下通りが若者文化の中心。日本初のクラブ「ピテカントロプス」も誕生。

1982
マハラジャ1号店オープン

オリエンタルで豪華な内装のマハラジャがオープン。バブル景気と共に高級ディスコブーム（第2次ディスコブーム）を牽引。「お立ち台」文化の原型が六本木スクエアビル3Fの『ギゼ』で誕生。

1980s後半
カーナンパの全盛期

車がステータスの時代、「格好良い車に乗っていればモテる」。千葉・幕張の美浜大橋は「ナンパ橋」と呼ばれ、週末には3km以上のナンパ待ち車列ができた。関東最大のカーナンパスポット。

1991
ジュリアナ東京オープン / バブルの絶頂

5月にジュリアナ東京オープン。平日1000人、週末2000〜3000人がフロアに。お立ち台・ボディコン・扇子が象徴。しかし既にバブルにはほころびが。1994年8月31日に閉店。

1990s前半
渋谷ストリートナンパの最盛期

渋谷がナンパの聖地として君臨。マルキュー前・公園通りが中心地。高校生〜20代前半のナンパ師が日夜繰り出した。チーマー文化、渋カジも同時代。95年頃にピークを迎え、徐々に衰退へ。

1993
ブルセラ・援助交際ブーム

宮台真司が『制服少女たちの選択』出版。援助交際を社会問題として可視化。1996年に流行語大賞入賞。出会いの商品化が議論に。

1994
ヴェルファーレ開業 / ディスコからクラブへ

六本木にヴェルファーレが開業（〜2007年）。ジュリアナ閉店後のナイトシーンを引き継ぐ。「ディスコ」が死語化し始め、小箱の「クラブ」が主流に。ナンパの場もクラブへ移行。

2000s
カーナンパの衰退 / ストナンスポットの絶滅

路上駐車の取り締まり強化・道路交通法改正でカーナンパが下火に。六本木・渋谷・池袋のストナン名物スポットも次々と消滅。車への関心低下・携帯電話の普及が追い打ち。

2003
スーパーフリー事件

早稲田大学のイベントサークル「スーパーフリー」による集団性的暴行事件。合コン文化の闇が社会問題化。主犯に懲役14年。

2011-16
風営法ダンス規制 → 改正

2011年以降、飲食店を装って客にダンスをさせた風営法違反で摘発が増加。クラブ文化存続の危機に。署名運動や議論を経て、2016年に改正風営法施行。照度10ルクス超のクラブが合法化。

オンライン: テレクラからマッチングアプリへ

1980s後半
テレクラの誕生

電話を通じて異性と出会う「テレフォンクラブ」が登場。風俗営業法の規制対象外だったため急成長。ナンパの「匿名性」「効率化」の原点。2002年の風営法改正で規制対象となり衰退。

1999
スタービーチ（スタビ）誕生

ガラケー向け無料メル友募集サイト「スター・ビーチ」がサービス開始。1日1,200万アクセス、月間4.5億アクセスを記録し、ガラケー時代の伝説的出会い系に。2008年の出会い系規制法施行（年齢確認厳格化）に対応できず2009年に閉鎖。

2001
出会い系サイトの時代

ワクワクメール（2001年）、PCMAX（2002年）開始。インターネット経由の出会いが一般化するも、犯罪被害も発生し「出会い系サイト規制法」制定へ。

2004
mixi誕生 → コミュニティ/オフ会文化

招待制SNS「mixi」がサービス開始。2006〜2009年に大ブームとなり、趣味コミュニティからオフ会→出会いへの流れが自然発生。「mixi＝出会える場所」のイメージが広まるも、未成年トラブルや援助交際が問題化。

2006
モバゲータウン誕生

ガラケー向けSNS+ゲーム「モバゲータウン」がサービス開始。半年で140万人、2008年に1000万人突破（国民13人に1人）。アバター・ミニメール・日記による交流が10〜20代を中心に爆発的人気。出会い目的での利用も。

2008頃
Skype掲示板の時代

「スカイプちゃんねるｗ」等のSkype ID共有掲示板が登場。テキスト→音声→ビデオ通話と段階を踏んで出会いに発展するパターンが定着。マッチングアプリ以前の「ネトナン」の原型。

PUA文化の輸入と恋愛工学

2005
『The Game』出版（米国）

ニール・ストラウスが PUA（Pick Up Artist）の世界を描いたノンフィクション『The Game』を出版。NYタイムズ1位のベストセラーに。PUA の概念が世界に広まる。ミステリーメソッド（オープナー→アトラクション→コンフォート→クロージング）が体系化。

2012
日本語訳『ザ・ゲーム』発売 / Pairs・Omiai 誕生

『ザ・ゲーム 退屈な人生を変える究極のナンパバイブル』として日本語版が発売。同年、日本初の本格的マッチングアプリ Omiai（2月）・Pairs がリリース。出会いのデジタル化が本格始動。

2014
タップル・with リリース / Tinder 日本上陸

マッチングアプリの三強が出揃う。Tinder の「スワイプ文化」が日本にも波及し、出会いのカジュアル化が加速。

2015
恋愛工学の爆発

藤沢数希が『ぼくは愛を証明しようと思う。』を出版。メルマガ「週刊金融日記」で恋愛工学を体系化。「非モテコミット」「ACSモデル」「フレンドシップ戦略」等の用語が界隈に浸透。ナンパ界隈がSNS展開に目覚め、Twitter上でのコミュニティ形成が加速。

講習文化からサロン文化へ

2013頃
ナンパ講習の黎明期

個人のナンパ師が1対1または少人数で実地指導する「ナンパ講習」が始まる。座学のみ、グループ講習、マンツーマンなど形態は様々。価格帯も数千円〜数十万円と幅広く、質のばらつきが問題に。

2015
オンラインサロンプラットフォームの拡大

Synapse（2012年〜）に続き、Salonde（2015年）、DMMオンラインサロン（2016年）が参入。ナンパ界隈でもオンラインサロン形式のコミュニティが形成され始める。

2017頃
まーぼー飯店（MBH）/ 講習文化の成熟

元No.1ホストの「まーぼー」が主宰するナンパ講習グループ「まーぼー飯店（MBH）」が活動。ナンパ歴18年・講師歴10年・受講者1000人超。メンバーが「2ヶ月で15,000声かけ」等のハードなストナン訓練を実施する体育会系スタイル。講習文化の代表格の一つ。

2018
Nanko設立 / サロン文化の本格化

のんちゃま主宰のナンパサロン「Nanko」が設立（後に600〜870人規模に成長）。クローズドなコミュニティで情報共有・仲間作り・講習を一体化。個人講習→サロン型への移行が加速。アツストサロン等も同時期。

2020
noteがナンパ系コンテンツを一斉BAN

2020年6月、noteが「稼げる系」「恋愛（主にナンパ術）」の情報商材コンテンツを一斉に削除。有料noteでナンパテクを販売していた層が打撃を受け、Tips・Brain等の別プラットフォームに移行。

Twitterナンパ界隈の形成と黄金期

2016-18
ナンパ界隈の黄金期

Twitter上に「ナンパ垢」が大量発生。地域別クラスター（🦁新宿・🐕渋谷・🦉池袋）が形成。ストナン・ネトナンの二大流派が確立。即報告・戦果報告が界隈のメインコンテンツに。コンサル・サロン・講習ビジネスも本格化。

2018
リアルナンパアカデミー（RNA）事件

ナンパ塾「RNA」の塾長が準強制性交等罪で逮捕（4度）。女性を泥酔させてホテルに連れ込む手法を「マニュアル化」し組織的に犯行。塾生に東京メトロ社員も。2020年に懲役13年。ナンパ塾の闇が社会に知られるきっかけに。

2019
中星マインドの台頭

中星一番が東大大学院中退後にナンパで月商2000万以上を達成。「女性に家を提供させる」「サンクコストをかけさせる」マインドセットが経典化。note累計約1億円売上。

コロナ禍と界隈の転換

2020
コロナ禍でストナン消滅 → アプリ全盛

緊急事態宣言で「不要不急のナンパ」が誕生。Zoom合コン・マスク詐欺など新ミームが続出。マッチングアプリが過去最高ユーザー数を記録。

2021
小田急線刺傷事件

8月、自称ナンパ師の男が小田急線で10人を刺傷。PUA→挫折→ミソジニーのパスが「インセル問題」として社会問題化。懲役19年。

2022
即画文化の全盛期 / 頂き女子

ベッド写真・音声の無断投稿が常態化。「頂き女子りりちゃん」が話題になり、PUAと頂き女子の「表裏一体」が議論に。港区女子・Photojoy量産型も定番ネタ化。

法規制の時代

2023.7
撮影罪 + 不同意性交罪の施行

同日施行の二つの法律がナンパ界隈に影響。即画（性的画像の無断撮影・公開）と、酔わせてのセックスが明確に犯罪化。「蛙化現象」「頂き女子りりちゃん逮捕」「杉崎PUAブレイク」も同年。

2024
相次ぐ問題と活動停止

ゆってぃYouTubeチャンネル削除（9月）→ スタナン一家・韓国遠征炎上で国際問題（10月）→ 孕柱メロ不同意性交で逮捕（11月）。アツストサロン即数捏造、黒足の一味も活動縮小。

2025
2025年の動向

のんちゃま引退（8年間のストナン講習終了）。Pairs・タップルが本人確認を厳格化。マッチングアプリぼったくり詐欺（トクリュウ、被害8000万円）が社会問題に。ロマンス詐欺急増（被害400億円）を受けデジタル庁がマイナンバーカード本人確認を要請。ゲンキジャパンが「コンプラ的に厳しい」と英国移住を発表。

PUA界隈ヒストリー とは？

PUA界隈ヒストリー は、PUA（Pickup Artist）/ ナンパ / マッチングアプリ界隈で流行したミームやネタを年ごとに整理する文化アーカイブです。 Twitter（X）や匿名掲示板、note、YouTubeなどで生まれたフレーズ・コラ画像・ネタを時系列で振り返ることができます。
